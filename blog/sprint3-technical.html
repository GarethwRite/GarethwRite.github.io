An analogy for the differences between html and css

Think of a person, html is the main content like the arms,body head and legs and css is the the visual styling of the person e.g. jeans, hat, jacket. So simply put html is the main content of a webpage and css is responsible for the layout and styling of the webpage.

Control flow and loops explained

control flow is basically as it is named, control flow is the order in which code is executed in javascript. Loops are a block of code that is executed a number of times until the loop ends or a certain output is reached depending on the code.

What is the DOM?

The DOM(document object model) is a representation of a web page in a tree like structure where objects or elements are displayed in a hierarchy starting at the root directory, it is a useful way to display and minipulate and access elements.

What are functions and what are they good for?

A function is a block of code made to prefom a specific task, they are very useful for altering a webpage after it has been loaded, making a website more interactive.
